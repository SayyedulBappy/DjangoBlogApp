# Blog Application

A blog application made with python django framework

## Features

- user authentication and authorization
- CRUD functionality for posts
- comment system on posts

## Description

Users can create account and add new posts to the site and update and delete their posts. Users can add comments to any post and delete comments

## What did I learn

- Basics of Django Framework
- Creating Models with Django ORM
- Using Function Based Views and Class Based Views
- url routing and template rendering
- user authentication and authorization
